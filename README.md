
# Labor-Free Kitchen

## Overview

Labor-Free Kitchen is a smart kitchen assistant designed to simplify food tracking and calorie estimation. The system combines computer vision, weight sensing, and temperature monitoring to automatically recognize food and provide users with nutritional information.

This project was developed as part of a multidisciplinary engineering project at New York University.

---

## Features

- Food recognition using Raspberry Pi Camera
- Weight detection using Force Sensitive Resistor (FSR)
- Temperature monitoring
- LCD user interface
- Raspberry Pi + Arduino communication
- Custom 3D enclosure

---

## Hardware

- Raspberry Pi 4
- Arduino Uno
- Camera Module
- FSR Sensor
- Temperature Sensor
- LCD1602 Display
- Servo Motor
